Build a complete Cosmetic Ingredient Detector Web App using HTML, CSS, JavaScript (frontend) and Django (backend). The app must be clean, modern, aesthetic, and have a strong vibe-code look: smooth animations, soft gradients, glassmorphism, rounded shapes, minimal and spacious layout.

APP PURPOSE

The app allows users to:

Upload an image OR scan using device camera

Extract text (ingredient list) using OCR (pytesseract)

Detect harmful cosmetic ingredients

Show why each ingredient is harmful in simple bullet points

Detect if the product is veg or non-veg (based on known animal-derived cosmetic ingredients)

Give a “SAFE”, “CAUTION”, or “HARMFUL” result

Store scans in database through Django admin

LIMITED SCOPE FOR NOW

➡️ Only detect cosmetic products — skin care, hair care, makeup.
